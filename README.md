Zomato Data Analysis Dashboard (Uttar Pradesh)

This project is a Power BI dashboard analyzing Zomato restaurant data across cities in Uttar Pradesh.

Key Insights

- City-wise restaurant distribution
- Average price comparison across cities
- Top 10 highest-rated restaurants
- Cuisine distribution

Tools Used

- Power BI
- Data Cleaning
- Data Visualization
